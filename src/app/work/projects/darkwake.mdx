## Overview

When I first started at Parana Games, I was tasked with rebranding Cash 8 Ball Pool from ground up. In addition to an aesthetic overhaul with better and more appropriate branding and marketing direction, the game's UX had to be reworked and several new features added.

Creating a beginner friendly, casual user interface was an interesting challenge for someone who never played pool.

One of my biggest challenges with this project was to create a UI that would be approachable for casual players, especially those new to the billiard game genre. Not being a pool game player myself and finding much social cash game interfaces to be confusing, I frequently choose to try completely fresh approaches to solve those problems, instead of copying what was already established on the market.
Together with the Art Director, we decided that the sport style should always take precedence in the visual and that the UI should be based on the competition mood. We also wanted to get rid of the casual, glassy style, which was in this game before.
